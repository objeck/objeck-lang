RegEx

Regular expression engine.

Support for following patterns:

  • any ‐ .
  • optional ‐ ?
  • zero or more repetitions ‐ *
  • one or more repetitions ‐ +
  • first single match ‐ ^
  • last single match ‐ $
  • (one|two|three) ‐ group
  • [start-end] ‐ range

Support for following special groups:

  • word ‐ \w
  • not word ‐ \W
  • digit ‐ \d
  • not digit ‐ \D
  • white space ‐ \s
  • not white space ‐ \S
  • repeat ‐ {least, most}
